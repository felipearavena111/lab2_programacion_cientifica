GUÍA DE LABORATORIO – SEMANA 2
Semana 2: Procesamiento de Datos con Funciones Avanzadas de Python
Nombre: Felipe Aravena 

Objetivo del Laboratorio
Aplicar funciones avanzadas de Python (lambda, map, filter y reduce) para procesar y
analizar datasets de gran tamaño, optimizando cálculos y diseñando pipelines de
procesamiento de datos.

Resultados de Aprendizaje
Al finalizar este laboratorio el estudiante será capaz de:
• Utilizar funciones lambda para definir operaciones anónimas.
• Aplicar map() para transformar datasets.
• Utilizar filter() para limpiar datos.
• Implementar reduce() para agregación de datos.
• Diseñar pipelines de procesamiento funcional para grandes volúmenes de datos.

Contexto del Problema
Una empresa de sensores industriales monitorea la temperatura de máquinas en una planta
de producción. Cada sensor genera miles de registros por hora. El objetivo del laboratorio
es procesar los datos para eliminar valores incorrectos, transformar unidades y calcular
métricas relevantes.
